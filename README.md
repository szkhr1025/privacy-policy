# privacy-policy

General-purpose privacy policy pages published via GitHub Pages.

## GitHub Pages

This repository is configured to publish from `main` branch (`/`).

Public URL:
- https://szkhr1025.github.io/privacy-policy/

## Usage

You can optionally specify the app name (and contact email) via query parameters:

- App name: `?app=Your%20App%20Name`
- Language: `?lang=en` or `?lang=ja` (default: both)
- Contact email: `?contact=you@example.com`

Examples:

- https://szkhr1025.github.io/privacy-policy/?app=My%20App
- https://szkhr1025.github.io/privacy-policy/?app=My%20App&lang=ja
- https://szkhr1025.github.io/privacy-policy/?app=My%20App&contact=mail@example.com
